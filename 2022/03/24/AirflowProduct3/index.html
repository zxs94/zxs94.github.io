AAirflow生产实践（三）操作

发表于 2022-03-24 分类于 Airflow

airflow操作

web页面操作

首页

首页展示DAGs,分为All,Active和Paused。其中Active下面的DAG是可以执行的DAG.

菜单栏

菜单栏分别如下：

DAGs为跳回首页。

Security包含用户、角色、权限等查看功能。

Browse包含Airflow包含的DAG Runs、Jobs、Task Instances等列表信息。

Admin包含一些配置等操作。其中Variables可以配置变量，在DAG代码中可以调用。Connection包含连接配置等Url地址。

Docs中可以查看一些官方文档、API接口等。更多详细内容可以在这里查看。

DAGS操作

DAGs首页列表中，从左到右依次为：

按钮：Active开关

DAG：显示DAG名字和标签

Owner所属者

Runs ：最近的DAG run状态

Schedule: 定时方式，None表示没有指定定时

Last Run：

Recent Tasks：最近的Tasks状态

Actions：操作按钮，分别为手动触发DAG,刷新DAG,删除DAG(删除不要轻易操作)

Links：一系列操作链接，鼠标放在按钮上可以出现操作键

依次如下：

Tree View（常用）

可以树状图形式查看具体DAG Task的执行状态，从左往右按时间顺序展示。

Graph View（常用）

流程图图形形式显示DAG任务执行状态。

点击对应Task的按钮，可以弹出Task操作内容：

可以按照下面选项，对Task执行Run、Clear、Mark Failed、Mark Success等操作。

也可以点击最上面一行，查看实例详情，该任务具体执行命令、log日志、该Task所有实例、过滤Task只保留该Task及其上游Task。

Calendar View

显示具体日期DAG的运行情况

Task Duration

任务Task执行时间统计

Task Tries

任务重试次数

Gantt 甘特图

Detais

DAG详细信息

Code

DAG代码
