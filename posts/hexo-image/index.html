Hexo图片引用

August 8, 2019 阅读时间：1 分钟
Hexo
Hexo

经过参考资料决定使用hexo-asset-image插件来实现博客的图片引用

使用流程 链接到标题

基本参照网上的配置方式

  1. 设置站点配置_config.yml:将post_asset_folder: false改为post_asset_folder: true

  2. 执行 npm install hexo-asset-image –save 装插件

  3. 执行hexo new [xxxx],生成xxxx.md和xxxx文件夹

  4. 把要引用的图片拷贝到xxxx文件夹中

  5. 使用 ![](xxxx/example.jpg) 来引用本地图片

踩坑 链接到标题

使用以上配置在项目中插入图片发现，页面引用时无法显示，查看生成的静态页面发现标签图片是
<img src="/.io//xxx.png"/>

经过一番查询找了一个解决办法、就是将博客根目录下的package.json中的插件版本换成0.0.1版本 原有版本为(1.0.0),编辑完成后 hexo clean hexo g 即可看到图片img标签变为 <img src="/2019/08/08/‘文章文件名称’/‘图片名称’.png"/>

降低插件版本后即可成功引用。
