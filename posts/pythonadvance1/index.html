Python进阶之：*args,**kwargs的使用

July 31, 2019 阅读时间：1 分钟
Python
Python

维护模型接口代码的时候，发现在装饰器的函数内部和解析函数上面经常使用*args, **kwargs,这两个参数。

这两个参数到底代表什么呢？

1. *args 链接到标题

这个参数主要表示一个系列的参数，元素名前面带 *并不是指代指针，而是表示指向一系列的参数。

例举：

	*list = 'red', 'blue','green'

所以我们可以对函数传入一系列的参数，而只定义一个参数。这样做的好处就在于当函数参数个数未定的情况下，我们可以使用这种参数声明，给我们的编程带来便捷，特别是在参数确定不了的情况下。

	def test(*args):
	    for i, item in enumerate(args):
	        print(i, item)
	
	print("test:")
	test('red', 'blue', 'green')

向函数传递三个参数，输出结果：

test:
0 red
1 blue
2 green
